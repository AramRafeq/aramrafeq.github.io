سەرەتایەک بۆ نیشانەکاری HTML

by ئارام
published on September 15, 2016

وانه‌ی یه‌كه‌م

له‌م وانه‌یه‌ سه‌ره‌تایه‌ك وه‌ر ده‌گرین  چیمان پێویسته‌ بۆ نوسینی تاگ و كردنه‌وه‌ی لاپه‌ره‌كانمان له‌سه‌ر زمانی html

وانه‌ی دووه‌م

له‌م وانه‌یه‌دا باسی هێدینگ و په‌ره‌گراف ده‌كه‌ین له‌ زمانی html

وانه‌ی سێیه‌م

له‌م وانه‌یه‌دا باسی تێكست فۆرمات ده‌كه‌ین بۆ ده‌ست كاری كردنی تێكسته‌كان له‌ زمانی html

وانه‌ی چواره‌م

له‌م وانه‌یه‌دا باسی تێكست فۆرمات ده‌كه‌ین بۆ ده‌ست كاری كردنی تێكسته‌كان به‌شی دووه‌م له‌ زمانی html

وانه‌ی پێنجه‌م

له‌م وانه‌یه‌دا باسی كۆمێنت و برێك لاین ده‌كه‌ین له‌ زمانی html

وانه‌ی شه‌شه‌م

له‌م وانه‌یه‌دا باسی وێنه‌ ده‌كه‌ین و چۆنیه‌تی مامه‌ڵه‌  كردن له‌گه‌ڵ وێنه‌ له‌ زمانی html

وانه‌ی حه‌فته‌م

له‌م وانه‌یه‌دا باسی لینك ده‌كه‌ین و چۆنیه‌تی مامه‌ڵه‌  كردن له‌گه‌ڵ لینك له‌ زمانی html

وانه‌ی هه‌شته‌م

له‌م وانه‌یه‌دا باسی هێد ده‌كه‌ین له‌ زمانی html

وانه‌ی نۆیه‌م

له‌م وانه‌یه‌دا باسی لیست ده‌كه‌ین له‌ زمانی html

وانه‌ی ده‌یه‌م

له‌م وانه‌یه‌دا باسی ته‌یبڵ ده‌كه‌ین له‌ زمانی html

وانه‌ی یازده‌

له‌م وانه‌یه‌دا باسی ته‌یبڵ ده‌كه‌ین  به‌شی دووه‌م له‌ زمانی html

وانه‌ی دوازده‌

له‌م وانه‌یه‌دا باسی ته‌یبڵ ده‌كه‌ین به‌شی سێیه‌م له‌ زمانی html

وانه‌ی سێزده‌

له‌م وانه‌یه‌دا باسی فۆرم ده‌كه‌ین له‌ زمانی html

وانه‌ی چوارده‌

له‌م وانه‌یه‌دا باسی فۆرم ده‌كه‌ین به‌شی دووه‌م له‌ زمانی html

وانه‌ی پازده‌

له‌م وانه‌یه‌دا باسی دروست كردنی فۆرمێكی جوان ده‌كه‌ین له‌ زمانی html

وانه‌ی شازده‌

له‌م وانه‌یه‌دا باسی چۆنیه‌تی

نوسینی كوردی له‌ ده‌سته‌راسته‌وه‌

ئای فره‌یم

ره‌نگ بۆ تێكست و باكگراوند

دانانی وێنه‌ بۆ باكگراوند

بلۆك ئاند انلاین

ده‌كه‌ین له‌ زمانی html

وانه‌ی حه‌ڤده‌

له‌م وانه‌یه‌دا باسی ئینتینیتس ده‌كه‌ین له‌ زمانی html

وانه‌ی هه‌ژده‌

له‌م وانه‌یه‌دا باسی دیڤ  ده‌كه‌ین له‌ زمانی HTML(لەگەڵ کەمێک لە css بۆ رێکخستن و نیشاندانی شێوازی کارکردن)

تێبینی: جیا لە کۆرسەکانی تر وانەکانی ئەم کۆرسە رۆژانە بڵاو دەکرێنەوە
