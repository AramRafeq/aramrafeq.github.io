by ئارام
published on September 21, 2017

پێشەکی

لەم زنجیرە فێرکاریەدا باس لە رێگای دروستکردنی QR Code دەکەین لە سفرەوە پاشان ھەوڵ دەدەین کۆدێک بنوسین کە بتوانین QR Code پێ دروست بکەین بێگومان ئەم کردارانە دەکەین بەبێ بەکارھێنانی ھیچ لایبرەریەک یان ھیچ یارمەتیدەرێک ئەمە لەگەڵ ئەوەی کۆمەڵێک بابەتی بیرکاری و ئەلگۆریثم باسدەکەین کە پێوسیتە بۆ دروستکردنی QR Code ئەنجامدانی ئەم کارە ئاسان نیە لەبەر کەمی کاتم و گرانی بابەتەکە لەوانەیە ھەندێک لە بابەتەکان زۆر بەجوانی روون نەبن یان بە یەک جار تێنەگەیت ھەڵبەتە لەوانەشە کە نەتوانین پرۆگرامەکە بە تەواوی دروست بکەین بەڵام ئەمانە ئەگەری کەمن و ھەوڵ دەدەم بە سادەترین شێوە بابەتەکان شیبکەمەوە.
